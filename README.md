# Spawnocalypse

# Core Gameplay

First-person movement with jumping and crouching

Nerf-style projectile shooting

Enemies continuously spawn and track the player

Player health and game-over screen

# What I Added / Improved
Added jump action

Added crouching animation

Switched to Nerf dart projectile

Implemented enemy following AI

Added continuous enemy spawning

Added health system + game over screen



# Directions

Use WASD to move and the Mouse to look around.

Press Space to jump.

Press V to Crouch

Left Click To Shoot The Bullets
